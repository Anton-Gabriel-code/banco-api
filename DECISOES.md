# Decisções Tomandas:
 ## Decisões de Arquitetura:
>O spring boot foi utilizado, pois facilita a criação de uma API REST e tem suporte JPA.
>
>O SQlite foi utilizado no projeto por ser simples de utilizar e não tem a necessidade de instelar um servidor separado, o que acaba vindo a calhar por ser um projeto pequeno.
>
>A arquitetura do projeto foi sepoarado em camadas, seguindo o padrão MVC, para que o projeto fique mais organizado e mais fácil de fazer manutenção
